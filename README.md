#NodeMCU RFID API in MERN
This project is about NodeMCU IoT Board. You can send your RFID history datas and card ids and also you can monitorize in a list.

Requirements:

NodeMCU Board
- Wires
- USB Cable
- RFID Card Reader
- RFID Cards
- Two LEDs (should has different colors)


You can run it with:
`
npm run dev
`
